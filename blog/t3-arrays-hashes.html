Array is ordered collections of data. It looks like this: [ ] When you make an empty array, just type "array_name = []" in your text editor.

When call a specific data, we can use index. From the beginning, index 0, index number increments by 1.

array = ['Air', 'Bread', 'Cookie', 'Desk'] array[0] = 'Air' array[1] = 'Bread' array[2] = 'Cookie' array[3] = 'Desk'

Also, a negative index is used to call data from the end of the array. For example,

array = ['Air', 'Bread', 'Cookie', 'Desk'] array[-1] = 'Desk' array[-2] = 'Cookie' array[-3] = 'Bread' array[-4] = 'Air'

If you want to select the last item in the array and do not know how many items are in the array, just use a negative index.

In other words, you cannot use index to pull out specific data from a hash. Creating a new empty hash is simple: "hash_name = {}" When you want to insert a certain pair in the hash, simply type "hash_name [key] = value".
