What is Enumerable?

Enumerable is one of the Ruby's most useful modules. According to the ruby-doc webpage, an enumerable is

[a] mixin [that] provides collection classes with several traversal and searching methods, and with the ability to sort. The class must provide a method each, which yields successive members of the collection. If Enumerable#max, #min, or #sort is used, the objects in the collection must also implement a meaningful <=> operator, as these methods rely on an ordering between members of the collection. from: ruby-doc

In plain English, an enumerable is an array-like, which we can order each element and iterate through the elements. Due to its character, there are many enumerable-specific methods such as map, cycle, and group_by available. We are going to see "#map" today.



How to use #map?

Map returns a new array, not changing the original collection of the element. When we call map method like below, assuming the collection name is numbers, each element passes the block and the results of the block are added to a new array:

numbers.map{|obj| block}
