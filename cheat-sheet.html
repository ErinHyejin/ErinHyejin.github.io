What is prototype in JavaScript?

Like most object-oriented languages, objects are the basic structure used to store data. The object can contain any number of properties (which contain values that characterize the object), and methods (which perform actions that can modify the object and its properties).

A prototype is way to create new objects. These new objects can be new versions of objects already defined by Javascripts, or they can be entirely new prototypes defined by the user. To see how to create a new prototype, and modify its properties and methods, see the sections below.

How to create prototypes?

Creating a prototype is as simple as defining a new function (called the object-constructor function). This simply involves typing the ‘function’ keyword, your function name, and populating the function with your properties and methods.

To actually create an object from this prototype, simply use the ‘new’ keyword.

Here are examples to clearly illustrate this:

function coder(name, skill, experience) { this.name = name; this.skill = skill; this.experience = experience; } var firstCoder = new coder(“Erin”, “ruby”, “1 year”) var secondCoder = new coder(“Sahil”, “JavaScript”, “2 year”)

How to add properties?

If your object already exists, you can easily add properties to that object. Simply type in the object name, followed by the dot operator, followed by the name of some new property you want to add, and set it equal to whatever value that property should have.

Here is an example to illustrate:

secondCoder.gender = “male”

It adds the property “gender” to the secondCoder only, not other objects. For example, firstCoder does not have the property “gender”.

How to add methods?

If your object already exists, you can easily add methods to that object. Simply type in the object name, followed by the dot operator, followed by the name of some new method you want to add, and set it equal to whatever function that method should perform.

Here is an example to illustrate:

firstCoder.introduction = function () { return this.name + "has " + this.experience + “ experience in ” + this.skill + “.”; };

*note: If you want to modify the prototypes themselves, you should use the ‘prototype’ property to add new properties or methods. If you do this, any objects created from this modified prototype will inherit these new properties or methods.

coder.prototype.gender = “male” coder.prototype.introduction = function () { return this.name + "has " + this.experience + “ experience in ” + this.skill + “.”; };
