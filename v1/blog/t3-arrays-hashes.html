Array is ordered collections of data. It looks like this: [ ] When you make an empty array, just type "array_name = []" in your text editor.

When call a specific data, we can use index. From the beginning, index 0, index number increments by 1.

array = ['Air', 'Bread', 'Cookie', 'Desk'] array[0] = 'Air' array[1] = 'Bread' array[2] = 'Cookie' array[3] = 'Desk'

Also, a negative index is used to call data from the end of the array. For example,

array = ['Air', 'Bread', 'Cookie', 'Desk'] array[-1] = 'Desk' array[-2] = 'Cookie' array[-3] = 'Bread' array[-4] = 'Air'

If you want to select the last item in the array and do not know how many items are in the array, just use a negative index.

What is you want to add items in an array? It is simple! array.push item_name or array[specific_index] = item_name

Though array is useful in many ways, it is not good for searching an item from tremendous amount of data. To search the item, we should go over the array one by one, which takes long time. Is there any alternative way? Yes, it is hash!

hashes { } A hash is a collection of key-value pairs and the pairs are not ordered lists. In other words, you cannot use index to pull out specific data from a hash.

Creating a new empty hash is simple: "hash_name = {}" When you want to insert a certain pair in the hash, simply type "hash_name [key] = value".
